# PHP with PHPUnit

## Instalation
First install [pear](http://pear.php.net/manual/en/installation.getting.php).

`$ pear config-set auto_discover 1`

`$ pear install pear.phpunit.de/PHPUnit`

## How to run a test

`$ phpunit tests`
